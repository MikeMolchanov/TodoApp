Todo App (iOS)

Простое приложение для управления задачами, написанное на Swift + UIKit.

Возможности
	•	добавление задач
	•	редактирование задач
	•	удаление задач
	•	отметка задачи как выполненной
	•	автоматическая сортировка выполненных и невыполненных задач
	•	поиск задач
	•	отображение empty state (если задач нет или ничего не найдено)
	•	анимация изменения статуса задачи
	•	сохранение данных через UserDefaults + Codable

Технологии
	•	Swift
	•	UIKit
	•	UITableView
	•	UISearchController
	•	UserDefaults
	•	Codable

Архитектура

Проект использует простую архитектуру MVC:
	•	ViewController — управление интерфейсом
	•	TaskStorage — работа с данными
	•	Task — модель данных
	•	TaskCell — кастомная ячейка таблицы
